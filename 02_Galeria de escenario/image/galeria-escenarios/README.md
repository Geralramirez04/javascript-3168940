# Galería de Escenarios

Este proyecto es una galería interactiva que muestra diferentes escenarios a través de una serie de diapositivas y miniaturas. Los usuarios pueden navegar por las escenas y ver detalles de cada una de ellas.

## Estructura del Proyecto

El proyecto tiene la siguiente estructura de archivos:

```
galeria-escenarios
├── src
│   ├── index.html          # Documento HTML principal
│   ├── app.js              # Lógica principal de la galería
│   ├── styles.css          # Estilos para la galería
│   ├── components
│   │   ├── thumbnail.js    # Componente para las miniaturas
│   │   └── slide.js        # Componente para las diapositivas
│   ├── data
│   │   └── scenes.json     # Datos de las escenas
│   └── utils
│       └── dom.js          # Funciones utilitarias para manipulación del DOM
├── package.json            # Configuración de npm
├── .gitignore              # Archivos y directorios a ignorar por Git
└── README.md               # Documentación del proyecto
```

## Instalación

1. Clona el repositorio en tu máquina local:
   ```
   git clone <URL_DEL_REPOSITORIO>
   ```

2. Navega al directorio del proyecto:
   ```
   cd galeria-escenarios
   ```

3. Instala las dependencias:
   ```
   npm install
   ```

## Uso

1. Abre el archivo `src/index.html` en tu navegador para ver la galería en acción.
2. Navega a través de las miniaturas para ver diferentes escenas.
3. Utiliza las flechas del teclado o los botones de navegación para desplazarte entre las diapositivas.
